# 基础视频通话

七牛云为开发者提供了一系列 Sample Code 以展示如何在应用中快速简单地集成以下功能

- [一对一视频通话](#一对一视频通话)
- [多人视频通话](#多人视频通话)

## 基础功能展示

### 一对一视频通话

一对一视频通话 Sample Code 展示了以下功能的使用

- 加入 / 离开房间
- 发布本地音视频数据到房间
- 订阅单个远端音视频数据
- 动态启用 / 禁用本地音频数据发布到远端
- 动态启用 / 禁用本地视频数据发布到远端
- 切换前后置摄像头
- 常用分辨率、码率、帧率等编码配置 (代码中展示)
